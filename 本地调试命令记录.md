# 本地调试命令记录

## 获取原码

- 删除旧的前端代码
  ```bash
  rm -rf ./代码
  ```
- 下载最新的 gh-pages 分支中的编译后的代码
  - https://github.com/LHRUN/paint-board/tree/gh-pages
    ```bash
    curl -L -o gh-pages.zip https://github.com/LHRUN/paint-board/archive/refs/heads/gh-pages.zip
    ```
  - 解压到 `./代码`
    ```bash
    unzip gh-pages.zip && mv paint-board-gh-pages "代码"
    ```
- 创建远程文件本地化目录，把项目中的所有远程应用文件下载到此目录，确保项目完全离线可用
  ```bash
  mkdir -p ./代码/remote_files
  ```
